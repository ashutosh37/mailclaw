# Project Constitution: Mailchimp Clone

## Vision
To build a reliable, scalable, and modular email marketing platform on Supabase.

## Development Principles
1. **API-First**: Accessible via API and Supabase client.
2. **Data Isolation**: Strict multi-tenant data separation via RLS.
3. **Serverless Priority**: Use Edge Functions for compute-heavy sending logic.
4. **Clean UI**: Responsive dashboard with Tailwind CSS.
